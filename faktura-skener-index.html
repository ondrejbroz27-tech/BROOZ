Jak nastavit Google OAuth:
1. console.cloud.google.com → nový projekt → Enable Google Drive API
2. OAuth consent screen → External → přidej svůj email jako Test user
3. Credentials → OAuth 2.0 Client ID → Web application
4. Authorized redirect URIs: přidej URL této GitHub Pages stránky
5. Zkopíruj Client ID sem → Uložit → Připojit Drive
